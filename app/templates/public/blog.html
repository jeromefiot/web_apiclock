Un environnement PYTHON sur mon mac

13 décembre 2013 Python Mac

Très bien ca mais c'est quoi l'idée et ca sert à quoi ?

  • Permet de créer un "espace", qu'on apellera dorénavant "environnement", dédié à son projet python sur son mac qui n'interfère pas avec le système du mac (si vous faites plein de conneries dedans, ce n'est pas un problème !)
  • Cet environnement peut avoir ses propres modules (des programmes unqiuement actif dans cet environnement... sans conséquence pour le système)
  • Il peut y avoir plusieurs environnements sur une machine avec des versions différentes de python par exemple.
Parfait donc pour tester un projet... voyons comment faire !
